# 教师工作量计算系统

一个用于统计和计算教师工作量的Web应用程序，支持Excel导入导出、学科系数管理和多学期数据统计。

## 功能特点

### 1. 教师信息管理
- 添加、编辑、删除教师信息
- 支持多学期数据管理
- 自动计算课时量（节数 × 学科系数）
- 按学期筛选查看数据

### 2. 学科系数管理
- 预设常见学科系数（语文、数学、英语等）
- 支持自定义添加学科和系数
- 一键重置为默认系数
- 实时更新课时量计算

### 3. Excel导入导出
- **导入功能**：从Excel文件批量导入教师数据
  - 支持的列名：教师姓名/姓名、学期/学年学期、科目/学科、节数/课时
  - 自动识别不同的列名格式
  - 支持多种学期格式（如"2024-2025学年第一学期"、"2024-2025-1"等）
  
- **导出功能**：将所有数据导出为Excel文件
  - 包含教师姓名、学期、科目、节数、学科系数、课时量
  - 自动生成带日期的文件名

### 4. 统计报表
- 学期统计：显示每个学期的教师数和总课时量
- 教师工作量排名：按工作量排序显示前10名教师
- 学科统计：显示各学科的教师人数和总课时量
- 按学年筛选统计数据

### 5. 工作日计算
- 自动计算学期工作日天数
- 排除周末（周六、周日）
- 支持不同学期的起止日期

## 使用方法

### 快速开始

1. 打开 `index.html` 文件，在浏览器中运行
2. 首次使用时，系统会加载默认的学科系数设置

### 添加教师数据

**方法一：手动添加**
1. 点击"添加教师"按钮
2. 填写教师姓名、选择学期、科目和节数
3. 点击"保存"

**方法二：Excel导入**
1. 准备Excel文件，包含以下列：
   - 教师姓名（或"姓名"）
   - 学期（或"学年学期"）
   - 科目（或"学科"）
   - 节数（或"课时"）
   
   示例格式：
   ```
   教师姓名  | 学期                | 科目 | 节数
   张三      | 2024-2025学年第一学期 | 数学 | 12
   李四      | 2024-2025-1         | 语文 | 10
   ```

2. 点击"导入Excel"按钮
3. 选择Excel文件
4. 系统自动解析并导入数据

### 管理学科系数

1. 切换到"学科系数"标签页
2. 点击"添加学科"添加新学科
3. 点击"编辑"修改学科系数
4. 点击"删除"移除学科
5. 点击"重置默认系数"恢复初始设置

### 查看统计报表

1. 切换到"统计报表"标签页
2. 选择要查看的学年
3. 查看各类统计数据

### 导出数据

1. 在"教师管理"标签页
2. 点击"导出Excel"按钮
3. 系统自动下载包含所有数据的Excel文件

## 数据存储

- 所有数据存储在浏览器的localStorage中
- 关闭浏览器后数据不会丢失
- 清除浏览器缓存会删除所有数据

## 学期格式说明

系统支持以下学期格式：
- `2024-2025-1`：2024-2025学年第一学期
- `2024-2025-2`：2024-2025学年第二学期
- `2024-2025学年第一学期`
- `2024-2025第一学期`
- 等多种变体

## 课时量计算公式

```
课时量 = 节数 × 学科系数
```

例如：
- 数学老师每周12节课：12 × 1.3 = 15.6 课时量
- 体育老师每周10节课：10 × 0.8 = 8.0 课时量

## 注意事项

1. **多学期任教**：如果一位老师在不同学期任教不同科目，请分别为每个学期添加记录
2. **数据备份**：建议定期使用"导出Excel"功能备份数据
3. **学科系数**：修改学科系数后，所有使用该学科的课时量会自动更新
4. **Excel导入**：确保Excel文件的表头名称正确，否则可能导致导入失败
